# 🚀 Hackathon Game Starter

## Quick Start (Dev)

```bash
npm install
npm run dev
```

Share the **Network** URL from your terminal with your PM teammate — it'll look something like:

```
  ➜  Network: http://192.168.1.47:5173/
```

Your PM can open that URL on their laptop to see the app live. It auto-refreshes every time you save.

## Quick Start (PM)

Open the URL your dev shares with you. That's it. You'll see changes appear in real time as the game gets built.

## Git

git add -A && git commit -m "added feature X"
